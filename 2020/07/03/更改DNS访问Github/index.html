更改DNS访问Github

最近真的是被github给折磨疯了，搞得我心态有点崩，家里网稍微有一点不好就上不去，博客还是基于gitpage的，github上不去就么得办法上传博客。被逼急了，来做个笔记吧，以后肯定会经常用的上。

首先用站长工具 找到TTL值最小的那一个

就比如这个，我选择的就是这个（当然，如果以后访问不上了，可以更换，这个只是暂时的最优解）

然后在本地计算机这个路径

c:\windows\system32\drivers\etc

修改hosts文件，写入刚才的IP地址就OK了
