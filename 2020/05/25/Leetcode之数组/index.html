560. 和为K的子数组

给定一个整数数组和一个整数 k，你需要找到该数组中和为 k 的连续的子数组的个数。

示例 1 :

输入:nums = [1,1,1], k = 2
输出: 2 , [1,1] 与 [1,1] 为两种不同的情况。

说明 :

  1. 数组的长度为 [1, 20,000]。
  2. 数组中元素的范围是 [-1000, 1000] ，且整数 k 的范围是 [-1e7, 1e7]。

代码：

public class Solution {
    public int subarraySum(int[] nums, int k) {
        int count = 0;
        for (int start = 0; start < nums.length; ++start) {
            int sum = 0;
            for (int end = start; end >= 0; --end) {
                sum += nums[end];
                if (sum == k) {
                    count++;
                }
            }
        }
        return count;
    }
}

思路：1.（枚举法）固定数组的一端然后遍历求和

面试题 10.01. 合并排序的数组

给定两个排序后的数组 A 和 B，其中 A 的末端有足够的缓冲空间容纳 B。 编写一个方法，将 B 合并入 A 并排序。

初始化 A 和 B 的元素数量分别为 m 和 n。

示例:

输入:
A = [1,2,3,0,0,0], m = 3
B = [2,5,6],       n = 3

输出: [1,2,2,3,5,6]

说明: 

A.length == n + m

代码：

class Solution {
    public void merge(int[] A, int m, int[] B, int n) {
        int pa = 0, pb = 0;//定义AB数组的两个指针
        int[] sorted = new int[m + n];
        int cur;//定义缓存变量
        while (pa < m || pb < n) {//直到pa或pb两个指针运行到结尾时跳出循环
            if (pa == m)    //如果pa指针已经到了末尾，直接把B给缓存变量就OK，下面和这个一样
                cur = B[pb++];
            else if (pb == n)
                cur = A[pa++];
            else if (A[pa] < B[pb])//对数组进行排序，然后给变量赋值
                cur = A[pa++];
            else
                cur = B[pb++];
            sorted[pa + pb - 1] = cur;//存到一个排序数组里
        }
        for (int i = 0; i != m + n; ++i)
            A[i] = sorted[i];//遍历存到原数组中，结束
    }
}
