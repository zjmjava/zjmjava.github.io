JDBC基础——增、删、改、查

对数据库进行增删改

  1. 通过Connection对象创建statement
  2. statement（陈述） 是SQL语句的发送器，功能是向数据库发送SQL语句
  3. 调用它的executeUpdate(String sql)（汉译：执行更新），它可以发送DML 数据操纵语言（Data Manipulation Language） DDL 数据库模式定义语言(Data Definition Language)，返回值为int，意义为有几条数据被修改了。

代码：

Connection con = DriverManager.getConnection(url,username,password);//获取链接
Statement  stat = con.createStatement();//通过链接创建一个新的发送器对象
//下面就和正常的sql语句一样，注意双引号里面的sql语句末尾不要加分号
String sql = "update animal set number = 100 where name = 'dog'";
int i = stat.executeUpdate(sql);//执行语句会获得被修改的数据条数

对数据库进行查询

具体方法同上，但是查询时statement对象调用的是executeQuery（String sql）(执行查询)，会返回一个ResultSet（结果集）

代码：

String sql = "select * from student";
ResultSet rs = stmt.executeQuery(sql);

由于ResultSet的toString方法不是太友好，所以我们需要自行遍历输出，这就要用到ResultSet的next方法和get方法了

next方法大概类似与一个指针，它指向查询出来的每一行数据，并且它的返回值是一个boolean值。初始时指向FirstBefore（第一条数据之前），结束时指向LastBefore,这两个位置对应的返回值都是false。所以我们就可以通过while循环来遍历出每一条数据。

ResultSet的get方法提供了获取各种类型的数据，如getString、getObject，getInt，getDouble等等，但是常用的只有前两种。使用时，我们可以传进去columnIndex（int型的列号），或者是columnLable（String类型的列名）。

代码：

while(rs.next()){
            String id = rs.getString("id");
            String psw = rs.getString(2);
            String name = rs.getString("name");
            System.out.println(id + " " + psw+ " " + name);
        }

最后我们在资源使用完毕时要进行关闭，关闭时遵循倒关的原则（即先启动的资源最后关闭）

代码：

rs.close();
stmt.close();
con.close();
