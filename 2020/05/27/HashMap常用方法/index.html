HashMap常用方法及基础

  • HashMap通过hash算法排布存储Map中的键（key），HashMap也是最常用的图状数据结构，其存储的数据元素是成对出现的，也就是说每一个键（key）对应一个值（value）。（key不允许重复）
  • HashMap中的数据元素不是按照我们添加的顺序排布的，并且其内存模式也不是连续的，但是其key值的排布是根据Hash算法获得的，所以在数据元素的检索方面速度还是较快的。
  • HashMap不能直接装入迭代器，必须将HashMap的所有键key装入迭代器，再进行遍历，或者是使用Entry类，将所有数据元素转化为Entry的集合进行处理。
public class hashmapTest {
    public static void main(String[] args) {
        HashMap<Integer,Integer> hm = new HashMap<>();
        hm.put(0,1);//添加
        hm.put(1,2);
        hm.put(3,null);
        hm.put(null,5);
        hm.put(8,9);
        System.out.println(hm.remove(8));//删除key对应的value  返回value
        System.out.println(hm.get(1));//通过map的key值获取对应的value值   返回value
        System.out.println(hm.isEmpty());//false 判断map是否为空   返回值类型boolean
        System.out.println(hm.containsKey(null));// true 判断是否存在键    返回值类型boolean
        System.out.println(hm.containsValue(2));//true 判断是否存在值      返回值类型boolean
        System.out.println(hm.size());//获取map中包含键值对的总数      返回值类型int
        System.out.println(hm.keySet());//返回由Map中的所有键组成的集合。     返回一个Set
        Collection<Integer> c = hm.values();//获取Map中的全部值组成一个Collection对象。  返回一个collection
        System.out.println(c.toString());
        hm.clear();//清空map里的所有元素
    }
}
